# Glassitas 🍪

Galletas decoradas artesanales para eventos sociales en San Luis Potosí.

**Especialidad:** XV años, baby showers, bodas, cumpleaños y cualquier celebración.

## Stack
- **Web:** HTML/CSS vanilla → GitHub Pages
- **Redes:** Facebook Page + Instagram Business
- **Publicación:** Node.js + Facebook Graph API
- **Imágenes:** Fotos reales + IA (ComfyUI/Gemini)
- **Logo:** SVG vectorizado
- **Automatización:** Cron local / Docker

## Estructura
```
44-glassitas/
├── .agents/            # Contexto para agentes de IA
├── assets/             # Logos, fotos, plantillas, posts, videos
├── contenido/          # Calendario y textos de publicación
├── marketing/          # Estrategia, copys, prompts, respuestas
├── web/                # Landing page
└── scripts/            # Automatización
```
